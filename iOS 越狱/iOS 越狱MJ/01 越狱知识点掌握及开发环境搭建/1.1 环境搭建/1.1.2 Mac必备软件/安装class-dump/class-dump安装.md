- 步骤1：[下载完工具包](http://stevenygard.com/projects/class-dump/)
- 步骤2：将class-dump文件复制到Mac的/usr/local/bin目录，这样在终端就能识别class-dump命令了

    ```swift
    /usr/local/bin
    ```

- 注意：由于mac权限问题，现在只能放到localbin目录下

    ```swift
    /usr/local/localbin
    ```

### 方式2：brew安装

```swift
brew install class-dump
```
